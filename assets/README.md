# Assets 폴더

이 폴더는 게임의 에셋 (음악, 이미지 등)을 저장하는 곳입니다.

## 배경 음악 (선택 사항)

배경 음악을 추가하려면:

1. `bgm.mp3` 또는 `bgm.ogg` 파일을 이 폴더에 추가하세요.
2. 게임을 실행하면 자동으로 배경 음악이 재생됩니다.
3. 음악 파일이 없어도 게임은 정상적으로 실행됩니다.

### 무료 배경 음악 다운로드 사이트

- [FreePD](https://freepd.com/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/music/royalty-free/music.html)
- [Bensound](https://www.bensound.com/)
